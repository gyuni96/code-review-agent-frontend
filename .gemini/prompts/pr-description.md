# Pull Request Description Template

GitHub PR 생성 시 사용할 수 있는 상세 설명 템플릿입니다.

## Template Body

```markdown
## Overview

(이번 PR에서 변경된 핵심 내용을 한 문장으로 요약)

## Changes

- [x] (변경 사항 1)
- [x] (변경 사항 2)

## Related Issues

- Closes # (이슈 번호)

## Screenshots / Evidence (Optional)

(UI 변경 사항이 있을 경우 스크린샷이나 동작 영상 첨부)

## Checklist

- [ ] Self-reviewed the code.
- [ ] No lint errors or console warnings.
- [ ] Verified the changes locally.
```
